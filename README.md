# 租帮宝 - 多后台订单监控助手

租帮宝用于自动监控多个业务后台的“待处理订单”。发现新订单时，会通过企业微信/飞书提醒，并提供快速跳转入口，方便及时处理。

## 你能做什么

- 同时监控多个站点的待处理订单数量
- 新订单提醒：桌面气泡 + 企业微信/飞书消息（可选）
- 一键打开后台处理：双击列表或使用“操作”入口
- 关闭不退出：可最小化到托盘后台运行

## 普通用户怎么用

### 1) 首次启动：激活授权

- 打开软件后，按提示输入授权码完成激活
- 如授权已过期，会先提示“已过期”，再让你输入新的授权码
- 需要报备设备信息时，到“用户信息”页复制“设备机器码”给管理员

### 2) 添加要监控的站点

- 打开“站点管理”页，点击“添加站点”
- 常规情况下只需要填写：站点名称、登录地址、账号、密码、订单地址
- 如你的站点需要额外的页面定位配置（用于识别订单数量/跳转链接），交付人员会提供或协助配置
- 站点管理列表右键可以修改站点配置，监控状态只有是启用时才会监控该站点的订单数量，如果禁用则不会检查该站点的订单数量

### 3) 配置提醒方式（可选但建议）

- 打开“高级设置”
- 企业微信/飞书提醒：把群机器人提供的 Webhook 地址添加进去即可（支持多个）
- Webhook 地址怎么拿：
  - 企业微信：群聊 → 右上角设置(...) → 消息推送 → 添加 → 复制 Webhook
  - 飞书：群聊 → 群设置 → 机器人 → 添加机器人 → 自定义机器人 → 复制 Webhook
- 桌面提醒：勾选“开启桌面气泡通知”
- 运行策略：
  - 检查间隔：多久检测一次订单
  - 无头模式：隐藏浏览器窗口（一般在当天已成功登录一次、并且重启监控服务后开启才明显生效；开启后无法人工介入登录）
  - 夜间免打扰：设置静默时段，避免夜间打扰

### 4) 启动监控

- 回到“运行监控”页，点击“启动监控服务”
- 列表会显示每个站点的待处理数量、更新时间，以及可执行的操作入口
- 遇到需要人工登录/验证码时，点击“显示浏览器界面”，处理完再“隐藏/移出屏幕”

## 常见问题

- 显示 0 单：可能确实没有订单；也可能未登录/登录失效。可先点“显示浏览器界面”确认是否需要重新登录。
- 收不到通知：确认已添加 Webhook 地址；群机器人是否正常；高级设置里是否开启了桌面气泡通知。
- 关闭窗口后找不到软件：检查右下角托盘图标，右键可“显示主界面/退出”。
- 频繁提示需要人工登录：建议每个站点用独立账号密码登录；多处共用同一账号容易互相顶号，导致反复掉线。

## 管理员（授权管理后台）

管理员可在授权管理后台进行：

- 新建授权、设置有效期、设置最多可激活设备数
- 修改授权有效期、作废授权
- 查看授权绑定的设备列表
